# Box2D Tests

This repository aim to be a sandbox to try things with Box2d physic engine.

## Installation

You need to install:

- g++
- Box2D
- SDL2
- OpenGL

## Run

To run a test, use the following command:

```
make single MAIN=[file name without .cpp extension]
```

For instance:

```
make single MAIN=1.inConsole
```
